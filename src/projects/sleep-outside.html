SleepOutside

Overview

Sleep Outside is a collaborative e-commerce platform designed for outdoor enthusiasts to purchase camping products. Developed as a college project, it features a user-friendly interface where customers can browse camping gear, view detailed product descriptions, add items to their cart, and complete purchases.

Live Demo GitHub Repository

Technologies Used

HTML5
CSS3
JavaScript

Key Features

  • Product Catalog - Browse a variety of camping products organized by categories with advanced filtering options.
  • Shopping Cart - Add products to cart, adjust quantities, and see real-time price updates.
  • Checkout Process - User-friendly checkout flow with shipping and payment information collection.
  • Responsive Design - Optimized shopping experience across all device sizes, from mobile to desktop.
  • Product Details - Comprehensive product information including specifications, pricing, and availability.

Challenges & Solutions

Challenge

Creating a persistent shopping cart that maintained product selections even after the user navigated away from the page or refreshed the browser.

Solution

We implemented localStorage to save cart data directly in the user's browser, ensuring cart contents would persist across page refreshes and browser sessions.

// Sample code for cart persistence
function saveCart(cart) {
  localStorage.setItem('so-cart', JSON.stringify(cart));
}

function getCart() {
  const cart = localStorage.getItem('so-cart');
  return cart ? JSON.parse(cart) : [];
}

Challenge

Implementing responsive design for product listings that looked good on all device sizes while maintaining good performance.

Solution

We used CSS Grid and Flexbox for layout, combined with media queries to adjust the display based on screen size. This allowed us to create a fluid interface that adapted to different viewports.
