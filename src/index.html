Krazy Katz API

The Krazy Katz API is a college project designed to provide a RESTful interface for accessing and manipulating data stored in MongoDB. Leveraging the Swagger framework for API documentation and interaction, Krazy Katz simplifies the process of performing CRUD (Create, Read, Update, Delete) operations on MongoDB collections.
